# DOL-pancake / DOL烤饼机
本烤饼机用于快速生成高度还原 [Degrees of Lewdity](https://www.vrelnir.com/) 游戏原文的伪截图，为四处抄代码缝合而来，如有 bug，请多担待，欢迎提出 issue。

## 计划
由于勉强能使用.jpg，无限鸽，如有实际需求，将更快完成
- [ ] 插入文本后光标自动移到上面
- [ ] 连续插入同一种文本
- [ ] 生成特效文本正常的截图
- [ ] 生成长图
- [ ] 支持更多游戏内容
